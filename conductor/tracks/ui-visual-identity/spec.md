# Track Specification: UI & Visual Identity Overhaul

## Objective
Transform ShinKu's UI to be more modern, immersive, and interactive, distinguishing it from standard TachiyomiSY.

## Requirements
1.  **Dynamic Theme Overhaul**:
    *   Extract primary colors from manga cover art using Android Palette.
    *   Apply these colors dynamically to the Material 3 theme when viewing or reading a specific manga.
2.  **Redesigned Manga Detail Page**:
    *   Modern layout with better spacing.
    *   "Clean Background" mode where the cover art fades or blurs into the background.
3.  **Drag-and-Drop Library**:
    *   Interactive long-press to move manga between categories.
    *   Visual reordering of categories.

## Scope
- Theme engine modifications.
- `MangaScreen` UI redesign.
- `LibraryScreen` interaction improvements.
