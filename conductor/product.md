# ShinKu Product Definition

ShinKu (Crimson) is a modern, rebranded, and enhanced fork of the SY base, which itself is a feature-rich fork of Mihon (formerly Tachiyomi).
The goal is to provide an immersive, intelligent, and highly personalized manga-reading experience while maintaining a core that is up-to-date with upstream fixes.

## Key Features
- **Vibe Search (AI Discovery):** Natural language manga search powered by Google Gemini AI.
- **Reading Journey Card:** Beautiful, interactive statistics tracking chapters read and time spent.
- **Dynamic Theme Overhaul:** Immersive UI that shifts colors to match manga covers.
- **Smart Categorizer:** Automated library organization into progress-based categories.
- **Dropbox Integration:** Secure cross-device synchronization replacing Google Drive.
- **Dead Source Scanner:** Specialized tool to migrate from uninstalled extensions.
- **Failed Updates Migration:** Dedicated menu to handle refresh failures.
- **Library Update Speed Toggle:** Three specialized modes (Standard, Boost, Extreme).
- **Metadata Refresh Cooldown:** Enforced 7-day cooldown to save network data.
- **Retained Core Features:** Feed menu, manga recommendations, advanced filtering, and metadata editing.

## Target Audience
- Users who want the advanced features of the SY base with modern AI-powered discovery.
- Readers looking for a more immersive and aesthetically pleasing UI than standard Mihon.
- Users who value library organization and synchronization across multiple devices.
