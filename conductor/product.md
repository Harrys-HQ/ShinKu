# ShinKu Product Definition

ShinKu is a fork of TachiyomiSY, which itself is a fork of Tachiyomi (now Mihon).
The goal is to provide a personalized experience while staying up-to-date with Mihon's upstream fixes.

## Key Features
- Based on TachiyomiSY's unique features (custom categories, etc.)
- Upstream synchronization with Mihon for core fixes and features.
- Custom branding (ShinKu).

## Target Audience
- Users who want the advanced features of TachiyomiSY with the reliability of Mihon's core.
