# ShinKu Tech Stack

## Android Development
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose / Android XML (Legacy)
- **Dependency Management:** Gradle (Kotlin DSL)
- **Asynchronous Programming:** Kotlin Coroutines & Flow
- **Dependency Injection:** Injekt (Tachiyomi's DI)
- **Database:** SQLDelight / Room
- **Networking:** OkHttp, Coil (Image Loading)

## Build System
- Gradle 8.x
- Android Gradle Plugin (AGP)
